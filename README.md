## <span id="article-usage">♗ 使用（Usage）</span>
    1. 下载项目 ( Download )

    2. 如果你了解如何使用不同平台下的编译命令，你需要分别编译客户端源文件(bin/client/client.c)和服务端源文件(bin/server/server.c)，编译完之后运行即可。（可以开启多个客户端）

    3. 如果你不了解编译命令，你也可以使用 Clion 打开项目，分别运行客户端和服务端的源文件。
