## <span id="article-usage">♗ 使用（Usage）</span>
    1. 下载项目 ( Download )

    2. 如果你了解如何使用不同平台下的编译命令，那你使用编译命令编译 Sherk.c 源文件即可。( Compile )

    3. 如果你不了解编译命令，你也可以使用 Clion 打开项目，运行 Sherk.c 文件即可。( Run )
